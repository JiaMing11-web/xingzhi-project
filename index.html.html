article设计系统基础

设计系统（Design System）是一组可复用的组件和标准的集合，用于构建大量产品。


核心要素：

1. 设计原则 — 指导设计的价值观和理念

2. 组件库 — 可复用的 UI 组件

3. 设计规范 — 颜色、字体、间距等标准

4. 图标系统 — 统一的图标风格


设计原则：

• 一致性 — 保持界面统一

• 可用性 — 用户能轻松使用

• 可访问性 — 所有用户都能使用

• 响应式 — 适配不同设备

UI 设计系统基础

重点知识点B站

设计系统是一套完整的 UI 设计标准和组件库，由设计原则、颜色体系、字体规范、间距系统、组件库和图标系统组成。


设计原则是设计系统的核心指导方针，常见的包括：一致性、可用性、可访问性、反馈性等。


颜色体系定义了品牌色、功能色、中性色的使用规范，确保颜色的一致性和可访问性。


组件库包含了按钮、输入框、卡片等可复用的 UI 组件，每个组件都有明确的设计规范和使用指南。
